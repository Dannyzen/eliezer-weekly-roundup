## What is implementable now

The clearest moves you can make now are:
- make files and artifact manifests the shared state for long-running agent work
- attach contextual scene traces to durable memory writes instead of storing only flat propositions
- expose memory retrieval through search, citations, privacy controls, and progressive disclosure
- represent instruction authority explicitly instead of flattening all inputs into one prompt stream
